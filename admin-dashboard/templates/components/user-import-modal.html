To import multiple users at the same time, MarinTrace can create users from a CSV (spreadsheet) file. You should download our starter template below to make sure MarinTrace can read your file. It should open in your default spreadsheet application (like Excel or Numbers).



Simply fill out the columns for each user you would like to import and upload the completed CSV below. They will receive email invitations (valid for 48 hours) which they can use to create their accounts. Since this may take some time, you may not see all the users at once. Come back to this page shortly to see your imports!
